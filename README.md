# 🚀 Behailu's Professional Portfolio

A modern, responsive personal portfolio website built with **Python**, **Flask**, and **Bootstrap 5**. This site showcases my journey as a Full-Stack Developer and UI/UX Designer, featuring a dark-themed interface with custom interactive elements.

## ✨ Features
* **Dynamic Routing**: Built with Flask to handle pages like Projects and Contact.
* **Interactive UI**: Custom grayscale-to-color hover effect on the profile image.
* **Responsive Design**: Fully mobile-friendly using Bootstrap's grid system.
* **Contact System**: A functional form with Flask Flash messages for user feedback.

## 🛠️ Tech Stack
* **Backend**: Python, Flask
* **Frontend**: HTML5, CSS3, Bootstrap 5
* **Design**: Figma

## 🚀 Getting Started
1. **Clone the repository**: `git clone https://github.com/BehailuBerhanu/portfolioB.git`
2. **Install Flask**: `pip install flask`
3. **Run the application**: `python app.py`
